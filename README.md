# Agenda Lite Free (v1.0.0)

Agenda Lite es un plugin de reservas para WordPress: servicios, profesionales, disponibilidad, páginas públicas de reserva, panel administrativo, clientes, exportaciones y seguridad antiabuso.

---

## Funciones principales (Free)
- Reservas ilimitadas para operar sin límites de agenda.
- Servicios ilimitados para ofrecer todas tus atenciones o tipos de cita.
- Profesionales ilimitados para equipos pequeños o grandes.
- Frontend completo para captar reservas desde tu web (flujo claro y rápido).
- Página pública de servicios + ficha individual por servicio (listas para publicar).
- Buscador, filtros y orden para que el cliente encuentre rápido.
- Calendario administrativo general (para administradores).
- Calendario individual por profesional.
- Gestión de citas: ver, editar, cancelar y reagendar desde el panel.
- Historial de reservas (administrador y profesional).
- Historial de clientes agrupado por correo (con métricas y trazabilidad).
- Popup de historial con reservas clickeables al calendario/backoffice.
- Exportación CSV: reservas, clientes y analítica.
- Panel de analíticas para seguimiento comercial.
- Pagos manuales mediante transferencias y pago presencial.
- Seguridad antiabuso: límites por período, verificación por código y bloqueos.
- Integraciones de marketing: GA / GTM / Google Ads / Meta Pixel (opcionales).
- Protección antispam con reCAPTCHA (opcional).
- SEO-friendly para páginas de servicios (base).
- Personalización de colores.
- Widgets/bloques compatibles con Gutenberg, Elementor y WPBakery (editor estable).
- Shortcodes para incrustar el flujo donde quieras.
- Optimizado para móvil y escritorio.

---

## Requisitos
- WordPress 6.0+
- PHP 7.4+

---

## Instalación
1. Sube la carpeta `agenda-lite` a `/wp-content/plugins/`.
2. Activa el plugin desde **Plugins** en WordPress.
3. Entra a **Agenda Lite → Ajustes** y configura moneda, zona horaria y preferencias base.
4. Crea servicios, asigna profesionales y publica tus páginas/widgets/shortcodes.

---

## Servicios externos (opcionales)
Agenda Lite solo conecta con servicios externos si el administrador los habilita en Integraciones:
- reCAPTCHA v3
- Google Tag Manager / Google Analytics / Google Ads
- Meta Pixel

Detalle de endpoints y datos enviados: `readme.txt` → **Servicios externos**.
